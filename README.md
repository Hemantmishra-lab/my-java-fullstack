🚀 My Java Full Stack Journey
Welcome to my comprehensive Java Full Stack repository! This project is a structured collection of modules covering everything from Core Java to Advanced Spring Frameworks. It serves as a living documentation of my technical growth and coding standards.

📂 Project Structure
This is a Maven Multi-Module Project designed for better organization and scalability:

trumio-java-lang: Foundations of Java (OOPs, Collections, Exception Handling, Multi-threading).

hibernate-concept: Deep dive into ORM (Object Relational Mapping) using Hibernate, XML/Annotation configurations, and CRUD operations.

trumio-spring-core: Core Spring concepts like Inversion of Control (IoC), Dependency Injection (DI), and Bean Lifecycle.

trumio-spring-jdbc: Integrating Spring with JDBC for efficient database communication.

trumio-spring-orm: Seamlessly connecting Spring with Hibernate/JPA.

🛠️ Tech Stack & Tools
Language: Java (JDK 17+)

Frameworks: Hibernate 7.x, Spring Framework

Database: MySQL 8.0

Build Tool: Maven

IDE: IntelliJ IDEA / Eclipse

Version Control: Git & GitHub

Master Dependency Injection patterns in Spring.
